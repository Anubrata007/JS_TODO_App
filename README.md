# JS_TODO_App

## Running the Project Locally

To run the JS TODO App on your local machine, follow these steps:

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (version X.X.X or later)
- [npm](https://www.npmjs.com/) (comes with Node.js)

### Steps to Run the Project

1. **Clone the Repository**

   Open your terminal and run the following command to clone the repository:

   ```bash
   git clone https://github.com/Anubrata007/JS_TODO_App.git
   ```

   when you clone the project and open VSCODE and you need to install live server extensions to run the project in local host.
